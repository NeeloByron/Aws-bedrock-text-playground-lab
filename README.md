## 📋 Lab Overview
This repository documents my hands-on experience with **Amazon Bedrock Text Playground**, a fully managed service that provides access to foundation models from leading AI companies through a single API.

**Duration:** 30 minutes  
**AWS Region:** US East (N. Virginia) us-east-1

## 🎯 Learning Objectives
- Navigate the AWS Bedrock console
- Access and test foundation models through Text Playground
- Experiment with prompt engineering
- Understand how temperature settings affect model outputs
- Compare responses across different models
